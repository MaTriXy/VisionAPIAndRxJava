# Vision API with RxJava

RxJava gives us an opportunity to work and process things on io thread without any problem and nested task management.

Demo app uses,
- Google Mobile Vision API
- RxPermissions
- RxImagePicker
- ButterKnife
- Glide

`MainActivity` contains all the code.
